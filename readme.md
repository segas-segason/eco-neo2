# Ecoposolstvo — сайт ивент-пространства

Frontend-проект сайта для ивент-пространства **«Экопосольство»** в Зеленоградске.
Сайт представляет площадку для проведения мероприятий и содержит описание пространства, возможностей, вариантов рассадки, FAQ и контакты.

Проект реализован как современный лендинг с адаптивной версткой, интерактивными элементами интерфейса и анимациями.

## Основные возможности

- адаптивная верстка по макету Figma
- компонентная структура интерфейса
- интерактивное мобильное меню
- popup-система
- accordion для FAQ
- анимации интерфейса на GSAP
- интеграция Яндекс.Карт
- подключение Яндекс.Метрики
- сборка проекта через Vite

## UI компоненты

### В проекте реализованы переиспользуемые интерфейсные компоненты:

- stat-card — карточки статистики
- service-card — карточки типов мероприятий
- space-card — описание особенностей пространства
- faq-card — accordion с вопросами и ответами
- popup — модальное окно
- navigation menu — адаптивная навигация

## Адаптивность

Проект реализован по принципу mobile-first с использованием Tailwind responsive utilities.

### Поддерживаются основные брейкпоинты:

- mobile
- tablet
- laptop
- desktop
- large desktop

dist

## Установка и запуск

### Установить зависимости:

    npm install

### Запуск dev-сервера:

    npm run dev

### Сборка production версии:

    npm run build

### Предпросмотр сборки:

    npm run preview

## Интерактив

### В проекте реализованы:

- мобильное меню
- popup с информацией о запуске проекта
- accordion для FAQ
- интерактивные UI элементы
- анимации интерфейса

JS-логика построена на использовании data-атрибутов, что позволяет разделить стили и поведение компонентов.
