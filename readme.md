# Ecoposolstvo — сайт ивент-пространства

Frontend-проект промо-сайта ивент-пространства **«Экопосольство»** в Зеленоградске.

Сайт представляет площадку для проведения мероприятий и содержит описание пространства, возможностей площадки, вариантов рассадки гостей, раздел FAQ и контакты.

Проект реализован как адаптивный лендинг с интерактивными элементами интерфейса, анимациями и интеграцией внешних сервисов.

## UI компоненты

В проекте реализованы переиспользуемые интерфейсные компоненты:

- **stat-card** — карточки статистики
- **service-card** — карточки типов мероприятий
- **space-card** — описание особенностей пространства
- **faq-card** — accordion с вопросами и ответами
- **popup** — модальное окно
- **navigation menu** — адаптивная навигация

## Установка и запуск

### Установить зависимости:

    npm install

### Запуск dev-сервера:

    npm run dev

### Сборка production версии:

    npm run build

### Предпросмотр сборки:

    npm run preview
